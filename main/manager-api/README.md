# Giới thiệu dự án

manager-api Dự án này được phát triển dựa trên framework SpringBoot.

Khi phát triển sử dụng trình soạn thảo mã, khi nhập dự án, chọn thư mục `manager-api` làm thư mục dự án

# Môi trường phát triển
JDK 21
Maven 3.8+
MySQL 8.0+
Redis 5.0+
Vue 3.x
